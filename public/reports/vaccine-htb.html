Vaccine Machine

Máquina Linux que encadena acceso FTP anónimo con un archivo ZIP cifrado, cracking de contraseñas MD5, inyección SQL en un panel web para RCE vía COPY FROM PROGRAM de PostgreSQL, y escalada de privilegios abusando de permisos sudo sobre vi.

Escenario

Vaccine es una máquina Linux del Starting Point de HackTheBox con IP 10.129.12.212. Proporciona un servidor FTP con acceso anónimo que contiene credenciales protegidas en un archivo ZIP. El panel web de administración usa PHP con una base de datos PostgreSQL vulnerable a SQL injection que permite ejecución de comandos del sistema.

Cracking ZIP + MD5

El ZIP está protegido por contraseña — cracking con zip2john:

zip2john backup.zip > backup.hash
john backup.hash --wordlist=/usr/share/wordlists/rockyou.txt

unzip -P 741852963 backup.zip
  inflating: index.php
  inflating: style.css

# Hash MD5 en index.php
grep -i password index.php
if(isset($_POST['username']) && isset($_POST['password'])) {

Cadena de Ataque

Paso 1
FTP Anónimo → backup.zip
Acceso FTP anónimo descarga backup.zip con el código fuente del panel de administración.
Paso 2
ZIP Cracking → 741852963
John the Ripper encuentra la contraseña del ZIP. El código fuente revela un hash MD5 de contraseña.
Paso 3
MD5 Cracking → qwerty789
Hashcat rompe el hash MD5 del panel de administración: admin:qwerty789.
Paso 4
SQLi → COPY FROM PROGRAM → RCE
El parámetro de búsqueda del dashboard es vulnerable a SQL injection en PostgreSQL. COPY FROM PROGRAM permite ejecutar comandos del sistema.
Paso 5
dashboard.php → Cred PostgreSQL
El archivo de configuración web contiene la contraseña de postgres: P@s5w0rd!.
Paso 6
sudo vi → GTFOBins → root
Permiso sudo para vi sobre un archivo de config. Shell escape desde vi para obtener root.
