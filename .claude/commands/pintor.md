## PINTURA DECORATIVA - INTERIORES & EXTERIORES

### Preparación de superficies (lo más importante)
```
El 80% del resultado final depende de la preparación.

PAREDES INTERIORES:
1. Limpieza: quitar polvo, grasa, hongos (agua + lejía diluida 1:10)
2. Reparación de grietas:
   - Grietas finas (<2mm): masilla acrílica directa
   - Grietas medias (2-5mm): cinta de fibra de vidrio + masilla
   - Grietas estructurales (>5mm): consultar especialista
3. Lijado de reparaciones (lija grano 120 → 220)
4. Sellador/imprimador según sustrato:
   - Pared nueva: sellador acrílico diluido 1:1
   - Pared con humedad: sellador antihumedad
   - Cambio de color oscuro→claro: imprimador pigmentado
5. Masillado general (si se requiere acabado perfecto):
   - Masilla lista: para pequeñas imperfecciones
   - Masilla en polvo: para mayor extensión

MADERA:
1. Lijar a favor del grano (120 → 180 → 220)
2. Limpiar polvo con trapo húmedo
3. Sellador de madera o wash primer
4. Masilla de madera en nudos y uniones
5. Lijar sellador (220)
6. Aplicar pintura o barniz

METAL:
1. Remover óxido: lija, cepillo metálico, convertidor de óxido
2. Limpiar con desengrasante o acetona
3. Imprimador anticorrosivo (wash primer o epóxico)
4. Pintura de acabado
```

### Tipos de pintura y cuándo usar cada una
```
INTERIORES:
- Vinílica mate: habitaciones, bajo tráfico - fácil de aplicar
- Vinílica semimate: salas, comedores - balance lavabilidad/brillo
- Esmalte al agua semibrillante: baños, cocinas - alta lavabilidad
- Esmalte satinado: marcos, puertas - durabilidad

EXTERIORES:
- Elastomérica: fachadas con microfisuras, alta flexibilidad
- Acrílica exterior: buena durabilidad UV
- Impermeabilizante: losas, terrazas, azoteas
- Epóxica: pisos (garajes, talleres)

MADERAS:
- Barniz poliuretano: máxima protección, brillo alto
- Barniz alkídico: exterior, buena flexibilidad
- Aceite de linaza/teca: acabado natural penetrante
- Esmalte alkídico: máxima durabilidad, colores opacos

METALES:
- Esmalte anticorrosivo: protección básica
- Pintura epóxica: alta resistencia química
- Pintura en polvo (powder coat): durabilidad extrema
- Galvanizado en frío: protección adicional
```

### Cálculo de material
```
REGLA GENERAL:
1 litro de pintura cubre ≈ 10-12 m² (primera mano)
1 litro de pintura cubre ≈ 12-15 m² (segunda mano, sobre imprimado)

CÁLCULO:
Área total = (Largo × Alto × 2) + (Ancho × Alto × 2)
Restar: ventanas (~1.5 m²), puertas (~2 m²)

Ejemplo cuarto 4×3m, 2.5m alto:
Área = (4×2.5×2) + (3×2.5×2) = 20 + 15 = 35 m²
Menos 1 puerta + 2 ventanas = 35 - 2 - 3 = 30 m²
Litros necesarios (2 manos) = 30/10 × 2 = 6 litros

DILUCIÓN:
Primera mano: diluir 20-30% (más penetración)
Segunda mano: diluir 10% o no diluir
```

## TÉCNICAS DECORATIVAS

### Técnicas principales
```
ESPONJADO (sponging):
- Esponja marina o de mar
- Aplicar color base, secar, aplicar segundo color con esponja
- Movimiento irregular, girar la esponja entre aplicaciones
- Resultado: textura orgánica y profundidad

RASTRILLADO (dragging/combing):
- Base semimate seco + glaze de color
- Pasar rastrillo o peine metálico mientras está húmedo
- Movimiento recto y continuo
- Resultado: efecto madera, seda o tela

STUCCO VENEZIANO:
- Aplicar con llana de acero inoxidable en capas finas (2-3)
- Cada capa perpendicular a la anterior
- Pulir con llana limpia mientras está semi-seco
- Resultado: aspecto mármol pulido

EFFETTO LÉGNO (efecto madera):
- Base amarilla/ocre
- Aplicar glaze marrón con brocha gruesa
- Pasar grainer o brocha de vetar mientras húmedo
- Resultado: veta de madera convincente

CHALK PAINT (pintura a la tiza):
- Para muebles sin lijar ni imprimar (adhiere bien)
- Acabado mate opaco vintage
- Sellar con cera de acabado o barniz mate
```

## PINTURA INDUSTRIAL & ANTICORROSIVA

```
SISTEMAS DE PINTURA POR AMBIENTE:

Interiores secos (oficinas, vigas):
→ Imprimador alkídico + esmalte alkídico

Interiores húmedos (plantas de alimento, farmacéutica):
→ Imprimador epóxico + esmalte epóxico

Exteriores suaves (fachadas metálicas):
→ Wash primer + esmalte poliuretano

Exteriores agresivos (marina, química):
→ Sandblasting SA 2.5 + epóxico zinc-rico + epóxico + poliuretano

Pisos industriales:
→ Epóxico 2 componentes auto-nivelante (300-500 micras)

NORMAS DE REFERENCIA:
- SSPC (Steel Structures Painting Council)
- ISO 8501-1: preparación de superficie (Sa 1, 2, 2.5, 3)
- NACE International: corrosión
```

## PINTURA AUTOMOTRIZ

### Proceso básico
```
1. DESMONTAJE: quitar piezas o cubrir lo que no se pinta
2. PREPARACIÓN:
   - Desengrasado con silicona remover
   - Lijar para eliminar brillo (320 → 400 en húmedo)
   - Reparar golpes: masilla poliéster, fibra si hace falta
   - Lijar reparación (80 → 120 → 180 → 320)
3. IMPRIMACIÓN:
   - Aparejo (primer surfacer) 2-3 capas
   - Lijar en húmedo (600 → 800)
4. BASE DE COLOR:
   - Colores sólidos: pintura directa 2-3 capas
   - Metalizados/perlados: base + efecto metálico + barniz
5. BARNIZ (clear coat):
   - 2-3 capas con 10-15 min entre manos
   - Es el que da brillo y protección UV
6. PULIDO Y ABRILLANTADO:
   - Lijar con 2000-2500 húmedo (si hay defectos)
   - Pulidora con compound
   - Polish de acabado
   - Cera o sellador
```

### Seguridad en pintura
```
□ Mascarilla con filtro orgánico (solventes) o FFP2 (polvo)
□ Gafas de protección
□ Guantes de nitrilo
□ Ropa de trabajo / mono de pintor
□ Ventilación adecuada (abrir ventanas, extractor)
□ No fumar (solventes son inflamables)
□ Revisar fichas de seguridad (SDS) de los productos
```
