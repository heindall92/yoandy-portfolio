## TIPOS DE INFORMES

### INFORME EJECUTIVO (Executive Summary / Board Report)
```
ESTRUCTURA:
1. Portada: Título, fecha, destinatario, clasificación
2. Resumen Ejecutivo (1 página máximo):
   - Propósito del informe
   - Hallazgos clave (3-5 puntos)
   - Recomendaciones principales
   - Decisión requerida (si aplica)
3. Contexto / Antecedentes
4. Análisis y Hallazgos
5. Conclusiones
6. Recomendaciones con priorización
7. Próximos pasos y responsables
8. Anexos

TONO: Formal, directo, orientado a decisión.
LONGITUD: 2-10 páginas según complejidad.
```

### INFORME TÉCNICO
```
ESTRUCTURA:
1. Portada técnica (versión, autores, revisores)
2. Tabla de contenidos
3. Lista de figuras/tablas
4. Glosario y abreviaciones
5. Introducción y alcance
6. Metodología
7. Resultados y análisis
8. Discusión técnica
9. Conclusiones
10. Recomendaciones
11. Referencias bibliográficas
12. Anexos técnicos

TONO: Preciso, objetivo, basado en evidencia.
FORMATO: IEEE, APA, o estándar del cliente.
```

### INFORME FINANCIERO / DE NEGOCIO
```
ESTRUCTURA:
1. KPIs y métricas clave (dashboard visual)
2. Performance vs objetivos
3. Análisis de variaciones
4. Análisis de tendencias
5. Riesgos y oportunidades
6. Proyecciones
7. Plan de acción correctivo (si aplica)

MÉTRICAS COMUNES:
- Revenue, MRR/ARR, Churn, CAC, LTV
- EBITDA, ROI, ROE, margen bruto/neto
- Burn rate, runway (startups)
```

### INFORME ACADÉMICO / CIENTÍFICO
```
ESTRUCTURA (IMRyD):
1. Título y autores
2. Abstract (150-300 palabras)
3. Introducción + hipótesis
4. Materiales y métodos
5. Resultados
6. Discusión
7. Conclusiones
8. Agradecimientos
9. Referencias (APA 7, Vancouver, IEEE, Chicago)

NORMAS: APA, MLA, Chicago, Vancouver según campo.
```

### INFORME GUBERNAMENTAL / INSTITUCIONAL
```
ESTRUCTURA:
1. Membrete institucional oficial
2. Número de expediente/referencia
3. Asunto
4. Destinatario oficial
5. Antecedentes legales/normativos
6. Objeto del informe
7. Consideraciones y análisis
8. Conclusión o resolución
9. Firma y sello

TONO: Impersonal, formal, con referencias normativas.
```

## ESTILOS POR IDIOMA

### Inglés
- Executive Summary al inicio (bottom-line up front)
- Bullet points y headings claros
- Estilo: activo, conciso, orientado a acción
- Evitar: jerga innecesaria, párrafos largos

### Español
- Estilo más formal y elaborado que el inglés
- Uso de "usted" en contextos formales
- Párrafos más desarrollados son aceptables
- Concordancia y ortografía perfectas obligatorias

### Francés
- Muy formal, uso de "vouvoiement"
- Estructura lógica cartesiana (problema → análisis → solución)
- Introducción larga justificando el contexto

### Alemán
- Extremadamente preciso y estructurado
- Sin ambigüedades, cada afirmación respaldada
- Uso extensivo de tablas y listas numeradas

### Portugués (BR/PT)
- Adaptar registro según variante (Brasil más informal que Portugal)
- Concordância verbal y nominal correcta

## PLANTILLAS RÁPIDAS

### Email ejecutivo (1 página)
```
Para: [Destinatario]
De: [Remitente]
Fecha: [Fecha]
Asunto: [Tema específico]

SITUACIÓN ACTUAL:
[1-2 oraciones sobre el contexto]

PROBLEMA/OPORTUNIDAD:
[1-2 oraciones sobre lo que requiere atención]

ANÁLISIS:
[3-5 puntos clave con datos]

RECOMENDACIÓN:
[Acción específica recomendada]

PRÓXIMOS PASOS:
• [Acción 1] - Responsable - Fecha
• [Acción 2] - Responsable - Fecha

[Firma]
```
