## DIAGNÓSTICO AUTOMOTRIZ

### Proceso de diagnóstico sistemático
```
1. ESCUCHAR al cliente: ¿Cuándo ocurre? ¿Qué síntomas exactos?
2. REPRODUCIR el problema (si es posible)
3. INSPECCIÓN VISUAL: fugas, cables dañados, desgastes visibles
4. ESCANEO OBD-II: leer códigos DTC
5. PRUEBAS específicas según síntoma
6. REPARAR y verificar que el problema se resolvió
7. TEST DRIVE para confirmar
```

### Códigos OBD-II más comunes
```
P0xxx - Sistema de tren de potencia genérico
P1xxx - Específico del fabricante
B0xxx - Carrocería
C0xxx - Chasis
U0xxx - Red/Comunicación

CÓDIGOS FRECUENTES:
P0300 - Fallo de encendido aleatorio/múltiple cilindros
P0301-P0308 - Fallo de encendido cilindro específico
  → Revisar: bujías, cables, bobinas, inyectores, compresión

P0171/P0174 - Sistema muy pobre (lean) Banco 1/2
P0172/P0175 - Sistema muy rico (rich) Banco 1/2
  → Revisar: MAF, sonda lambda, fugas de aire, inyectores

P0420/P0430 - Eficiencia del catalizador por debajo del umbral
  → Revisar: catalizador, sondas O2, fugas de escape

P0101/P0102 - MAF (Mass Air Flow) fuera de rango
  → Limpiar o reemplazar MAF, revisar fugas de aire

P0340/P0341 - Sensor de posición del árbol de levas
  → Revisar sensor, cable, timing chain

P0500 - Sensor de velocidad del vehículo
P0715 - Sensor de velocidad turbina (transmisión)
P0740 - Circuito solenoide convertidor de par
```

### Diagnóstico por síntomas

#### Motor no arranca
```
¿Hay ruido al girar la llave?
├── NO → Batería descargada, fusible, relé de arranque, motor de arranque
│         Verificar: voltaje batería (>12.4V), conexiones terminales
└── SÍ (gira pero no arranca) →
    ├── ¿Hay chispa? → Prueba de bujía, bobina, módulo de encendido
    ├── ¿Hay combustible? → Presión de bomba (spec del fabricante)
    └── ¿Hay compresión? → Test de compresión (>150 PSI por cilindro)
```

#### Ruidos por ubicación
```
MOTOR:
- Cascabeleo arriba (frío): tensor de cadena de distribución
- Golpeteo inferior (caliente): bielas, cojinetes de cigüeñal
- Clic-clic constante: balancines (ajuste de válvulas)
- Silbido: fuga de vacío, cinturón serpentine

FRENOS:
- Chillido al frenar: pastillas desgastadas (indicador metálico)
- Chirrido + vibración: disco deformado (runout)
- Golpe al frenar: soporte del caliper flojo

SUSPENSIÓN:
- Crujido en baches: barra estabilizadora, bujes de control
- Ruido de rodamiento: rodamiento de rueda (cambia con giros)
- Traqueteo delantero: rótulas o cazoletas desgastadas

TRANSMISIÓN:
- Difícil meter marchas: embrague, cable, hidráulico
- Resbalamiento automática: nivel/calidad del ATF, solenoides
- Golpe al cambiar D→R: amortiguadores de motor
```

## MANTENIMIENTO PREVENTIVO

### Intervalos de mantenimiento (referencia general)
```
CADA 5,000-10,000 km:
□ Cambio de aceite de motor y filtro
□ Revisar nivel de líquidos (frenos, dirección, refrigerante)
□ Presión y estado de neumáticos

CADA 15,000-20,000 km:
□ Filtro de aire del motor
□ Revisar pastillas y discos de freno
□ Rotación de neumáticos

CADA 30,000-40,000 km:
□ Bujías (iridio: 100,000 km)
□ Filtro de combustible
□ Líquido de frenos (DOT 4: cada 2 años o 40,000 km)
□ Revisión de correas y mangueras

CADA 60,000-90,000 km:
□ Correa de distribución (o cadena según fabricante)
□ Bomba de agua (con la distribución)
□ Líquido refrigerante

CADA 100,000 km:
□ Revisión completa de suspensión
□ Bujías de iridio
□ Revisión de transmisión
```

## MECÁNICA BÁSICA - GUÍAS PASO A PASO

### Cambio de aceite
```
Materiales: aceite especificado (ver tapa motor), filtro de aceite,
llave de filtro, llave combinada, recipiente para aceite usado

1. Calentar motor 5 min (facilita el drenado)
2. Apagar y esperar 5 min
3. Quitar tapón de drenado (con recipiente debajo)
4. Quitar filtro de aceite
5. Instalar tapón con arandela nueva (torque: ~25-35 Nm)
6. Instalar filtro nuevo (lubricar junta de goma con aceite)
7. Agregar aceite nuevo (cantidad según manual)
8. Arrancar, revisar fugas, verificar nivel
```

### Cambio de pastillas de freno
```
1. Aflojar tuercas de rueda con el auto en el suelo
2. Elevar con gato y colocar soporte de seguridad
3. Quitar rueda
4. Aflojar y quitar pernos del caliper (no desconectar manguera)
5. Colgar el caliper con alambre (no jalar la manguera)
6. Quitar pastillas viejas
7. Comprimir émbolo del caliper (con compresor o tabla)
8. Instalar pastillas nuevas (verificar posición de las orejas)
9. Colocar caliper, torquear pernos según especificación
10. BOMBEAR freno varias veces antes de mover el auto
11. Verificar nivel de líquido de frenos
```

## VEHÍCULOS ELÉCTRICOS (EV)

```
DIFERENCIAS CLAVE vs combustión:
- Sin aceite de motor, sin bujías, sin correa de distribución
- Sistema de alta tensión: 400V-800V (PELIGROSO, requiere capacitación)
- Batería de tracción HV + batería auxiliar 12V
- Motor eléctrico: mantenimiento mínimo (rodamientos)
- Frenos regenerativos: pastillas duran mucho más

MANTENIMIENTO EV:
- Líquido de frenos: igual que ICE
- Neumáticos: desgaste más rápido por el par instantáneo
- Refrigerante de batería: según fabricante
- Filtro de habitáculo: igual
- Software updates: OTA en Tesla/rivian, concesionario en otros

SEGURIDAD HV:
- NUNCA tocar el sistema de alta tensión sin capacitación HV
- Cable naranja = alta tensión, mantener distancia
- En accidente: no cortar cables naranjas
```
